## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| cursor position → completion context helper | Misclassified cursor context can produce misleading or noisy completions |
| current-document analysis → completion response | Stale or over-broad symbol data can leak beyond the current document scope |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-04 | I | completion context detection | mitigate | Limit completion contexts to explicit DVML keywords and reference tokens, then test negative cases for unrelated cursor positions |
| T-04-05 | T | same-document declaration index | mitigate | Build the index only from the active document AST and reuse the cache produced by 04-01 instead of scanning imports or workspace files |
| T-04-06 | D | completion handler | mitigate | Serve completions from the existing analysis cache so requests do not reparse on every call |
</threat_model>
